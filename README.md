# 🏡 Airbnb Clone

A full stack **Airbnb Clone** web application built using **Node.js, Express, MongoDB, and Tailwind CSS**.
Users can create property listings, upload images, and explore available stays.

---

## 🚀 Features

* 🏠 Create property listings
* 📷 Upload property images
* 📄 View all listings
* 🗂️ MVC architecture
* ⚡ Fast UI with Tailwind CSS
* 📁 Image upload functionality

---

## 🛠️ Tech Stack

**Frontend**

* HTML
* CSS
* Tailwind CSS
* EJS

**Backend**

* Node.js
* Express.js

**Database**

* MongoDB

**Other Tools**

* Multer (Image Upload)
* Nodemon

## ⚙️ Installation

Clone the repository

```
git clone https://github.com/avnishsingh309/airbnb-clone.git
```

Go to project folder

```
cd airbnb-clone
```

Install dependencies

```
npm install
```

Run the server

```
npm start
```

Open in browser

```
http://localhost:3000
```
